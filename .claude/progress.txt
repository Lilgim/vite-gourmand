Learnings:
- Un PostgreSQL natif Windows occupe 5432 → conteneurs mappés 5433 (pg) / 27018 (mongo).
- Driver mongodb (bson) incompatible Bun sur Windows (node:v8 isBuildingSnapshot) → scripts utilitaires avec `node --env-file=.env` ; l'app Next tourne sur Node donc pas d'impact.

- Premier bind d'un port docker parfois refusé juste après un up raté (relais WSL pas libéré) → retry suffit.
