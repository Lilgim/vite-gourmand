Livrer l'ECF « Vite & Gourmand » (TP Développeur Web et Web Mobile) avant le **23 juillet 2026, 23:59 heure de Paris** : une application web complète pour un traiteur bordelais, réellement exécutée, testée, déployée, documentée et défendable à l'oral.

## Définition de « terminé »

- Toutes les exigences du sujet officiel sont tracées dans `REQUIREMENTS_MATRIX.md` et reliées à une preuve (code, test, document, sortie d'exécution).
- Lint, types, tests, build et parcours end-to-end passent réellement (sorties consignées dans `VERIFICATION.md`).
- Un clone propre s'installe depuis le README ; les scripts SQL fonctionnent sur une base vide.
- La copie officielle contient les trois liens (dépôt public, app déployée, gestion de projet) et les identifiants de démonstration, tous testés en navigation privée.
